# Hand Gesture Recognition using SVM
This is the final task from the Skillcraft Technology which the project implements a Hand Gesture Recognition system using Support Vector Machine (SVM).
The model classifies different hand gestures from the image data.
 ## Technology Used
 - Python
 - openCV
 - Numpy
 - Sci-Kit learn
 - Google colab
 - Github

   ## Features
   - Image preprocessing
   - Gesture Classification
   - Hand Gesture recognition
   - SVM machine learning model
  
     ## Gestures used
     - Thumbs Up
     - Palm
     - Victory sign
    
       ## Files Included
       - Hand_Gesture_Recognition.ipynb
       - Sample hand gesture images
       - Extracted dataset images from GoogleChrome ZIP File
